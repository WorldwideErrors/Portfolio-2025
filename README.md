# Portfolio Jeffrey van Tillo


## Inleiding
Ten behoeve van de studie HBO-ICT en het ontwikkelingsproces van de studie in het kader van Semester 3 Front-End Development heb ik een portfolio website gemaakt in een minimalistische stijl. Hierbij heb ik verschillende kleurpaletten uitgetest om te kijken wat er het best werkt voor de toegankelijkheid van de website. Ook heb ik gekeken naar verschillende webtrends.

## Design Filosofie
- Minimalistisch en modern
- Sterke focus op contrast en leesbaarheid
- Rustige samenstellingen en consistente witregels
- Gebaseerd op hedendaagse web- en portfolio trends

## Toegankelijkheid
Toegankelijkheid stond centraal tijdens het project, omdat ik wil duidden op het feit dat ik websites en apps wil bouwen die voor iedereen beschikbaar zijn.

### Gebruikte Tools
- Colorblindly
- Lighthouse Accessibility

### Implementaties
- Hoge contrastverhoudingen
- Toegankelijke kleurpaletten voor alle thema's
- Consistente en duidelijke typografie

### Tech Stack
- React
- Next.js
- Typescript
- Eslint

## Features
- Dynamische projecten
- Custom minimalistische UI-componenten
- Mobile-first ontwerp
- Geoptimaliseerde performance
