# Portfolio Jeffrey van Tillo

## Inleiding
Ten behoeve van de studie HBO-ICT en het ontwikkelingsproces van de studie in het kader van Semester 3 Front-End Development heb ik een portfolio website gemaakt in een minimalistische stijl. Hierbij heb ik verschillende kleurpaletten uitgetest om te kijken wat er het best werkt voor de toegankelijkheid van de website. Ook heb ik gekeken naar verschillende webtrends.

## Toegankelijkheid
Voor de toegankelijkheid heb ik gebruik gemaakt van een chrome extensie genaamd Colorblindly. Dit is een programma waarmee je verschillende kleurafwijkingen kan simuleren om zo te zien of het prettig te bekijken is voor mensen met oogafwijkingen. Verder ben ik zelf half-blind en heb ik dus enorme belangen bij een prettig te navigeren website met veel contrast en duidelijke lettertypen.

## Projecten
Projecten heb ik via een simpel aan te passen typescript bestand uit laten lezen. Dit omdat ik al enige ervaring heb met databases en mijn focus tijdens dit project vooral wou leggen bij Figma designs omzetten naar een website. De manier van data verwerking was voor mij dan ook een stuk minder interessant.
